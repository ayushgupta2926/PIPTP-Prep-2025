## ✅ Q1. Simple Increment with While Loop

### 🔹 Approach:
Increment variable `a` in a `while` loop until it's no longer less than 10.

### 🔸 Java Code:
```java
public class Q1 {
    public static void main(String[] args) {
        int a = 1;
        while (a < 10) {
            a = a + 2;
        }
        System.out.println("a = " + a);
    }
}
 Output:a = 11
